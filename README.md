# Agenda de Contatos em Python

Projeto simples de **agenda de contatos** desenvolvido em Python durante meus estudos no curso de **Segurança da Informação da Solyd**.

O objetivo do projeto é praticar lógica de programação, estrutura de dados e interação com o usuário no terminal.

## Funcionalidades

O programa possui as seguintes funções:

* Mostrar todos os contatos da agenda
* Buscar contatos
* Adicionar novos contatos
* Interação com o usuário através do terminal
* Importar e exportar arquivos CSV
* Editar contatos já existemtes 

## Tecnologias utilizadas

* Python

## Como executar o projeto

1. Instale o Python no seu computador
2. Clone este repositório:

git clone https://github.com/RicardoRRA/agenda-python.git

3. Entre na pasta do projeto

cd agenda-python

4. Execute o programa

python agenda.py

Este projeto foi desenvolvido com fins **educacionais**, para praticar conceitos básicos de programação em Python.
